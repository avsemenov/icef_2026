# Additional Materials

## Books
* [R for Data Science (2nd ed)](https://r4ds.hadley.nz/) -- the best free starter for data science in R from the creator of tidyverse and ggplot
* [Forecasting: Principles and Practice (3rd ed)](https://otexts.com/fpp3/) -- the best free starter for time-series forcasting in R from the legend of the field. Contains YouTube videos.
* [OpenIntro Statistics](https://www.openintro.org/book/os/) -- the best free starter for statistics and data analysis in social sciences with videos, slides and labs.
* [Posit Cheatsheets](https://rstudio.github.io/cheatsheets/) -- condensed information for all the key packages in Tidyverse
